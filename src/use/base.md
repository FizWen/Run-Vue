### 如何下载使用 runvue ui 组件库

#### 第一步  下载 npm包

```javascript
npm i runvue@1.0.2  --save      // 版本会有比较快速的升级
```

#### 第二步  文件中引入

```javascript
import RunVue from 'runvue'
import 'runvue/style.css'

Vue.use(RunVue)
```

#### 第三步  载 tpl or html 中使用

```html
<uiButton type="success"> test </uiButton>
```
